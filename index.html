Getting Started with Azure

Microsoft Azure is a leading cloud platform with services for compute, storage, databases, networking, analytics and AI. This page focuses on practical study steps and resources for Azure certification candidates (example: AZ-104, AZ-900, AZ-305).

Quick study checklist

  • Understand core cloud concepts (IaaS, PaaS, SaaS)
  • Hands-on labs: deploy a VM, create an App Service, and try Azure Functions
  • Use Microsoft Learn modules and the official exam skills outline
  • Take timed practice tests and review weak topics

Top recommended resources

  • Microsoft Learn — free role-based learning paths
  • Official exam guide & skills outline (from Microsoft docs)
  • Hands-on labs: Azure sandbox or free tier
  • Community study groups, practice exams, and flashcards

Pro tip

Schedule short, focused daily study periods (25–45 minutes). Do active recall: build a resource in Azure, then explain what you did out loud or write notes.

Exam-specific notes

AZ-104 (Azure Administrator) — high-level

  • Focus: managing identities, governance, compute/networking/storage.
  • Practice: create VMs, implement RBAC, configure NSGs, backups, and monitor resources.

AZ-900 (Fundamentals)

  • Focus: core cloud concepts, basic Azure services, pricing & support.

Always pair reading with labs — hands-on beats passive reading.
